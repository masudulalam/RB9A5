Important FAQ

What is the Document Object Model(DOM)?

DOM stands for Document Object Model. It is a programming interface that allows us to create, change, or remove elements from the document. We can also add events to these elements to make our page more dynamic.

How do you select an element from DOM?

To select an element from DOM using any method of document.getElementById() or document.querySelector()

What is event delegation in the context of the DOM, and why it is useful?

Event Delegation is basically a pattern to handle events efficiently. Event Delegation is a pattern based upon the concept of Event Bubbling. It is an event-handling pattern that allows you to handle events at a higher level in the DOM tree other than the level where the event was first received.

How do you manipulate an elements attributes and styles using the DOM?

The classList object only provide methods to change the class of an element. If you want to change other attributes like id, href, or src, you can use the setAttribute() method.
